# 🏨 Hostel Management System (C++ Console Application)

## 📌 Project Overview

The **Hostel Management System** is a console-based application developed in C++ that helps manage hostel operations such as room allocation, student records, and availability tracking.

This project is designed for beginners to understand Object-Oriented Programming (OOP) concepts and basic data handling using C++.

---

## 🚀 Features

* ➕ Add new rooms
* 📋 Display all rooms
* 👨‍🎓 Add student with room allocation
* 📄 Display all students
* 🔍 Search student by ID
* ❌ Delete student record
* 🛏️ View available rooms

---

## 🛠️ Technologies Used

* Language: **C++**
* Concepts:

  * Object-Oriented Programming (OOP)
  * Classes & Objects
  * Vectors (STL)
  * Menu-driven programming

## ⚙️ How to Run the Project

### 🔹 Step 1: Compile the Code

Use any C++ compiler like g++:

```
g++ main.cpp -o hostel
```

### 🔹 Step 2: Run the Program

```
./hostel
```

(On Windows, use `hostel.exe`)
